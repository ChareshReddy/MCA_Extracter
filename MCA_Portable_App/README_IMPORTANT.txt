=====================================================
IMPORTANT: HOW TO RUN WITHOUT FIREWALL/ANTIVIRUS ISSUES
=====================================================

When you send this folder to your clients via email or a download link, Windows automatically flags it as "downloaded from the internet" and might block it.

To prevent ANY firewall or security blocks, tell your clients to do this:

1. Right-click the downloaded .zip file BEFORE extracting it.
2. Click "Properties".
3. Look at the bottom of the "General" tab for a checkbox that says "Unblock".
4. Check the "Unblock" box and click "Apply" -> "OK".
5. Now extract the folder.

After extracting, the client can simply double-click the "MCA_Data_Extracter.exe" file to start the engine!
